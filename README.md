# Корпус
Для проекта использовался только корпус трейн из папки проекта.
# Выделение аспектов
Для выделения аспектов я использовала CRF модель, которая обучалась на train_split_reviews и train_split_aspects.
# Оценка тональности
Для оценки тональности я использовала библиотеку dostoevsky. Для оценки я брала аспектов я брала до 7 токенов справа от аспекта и до 7 токенов слева.
# Файлы
dev_pred_asp_no_sent - только предсказанные аспекты без оценки
dev_pred_aspects - предсказанные аспекты и их эмоциональная окраска
dev_pred_cats - общая оценка категорий
crf_model.sav - сохраненная модель
